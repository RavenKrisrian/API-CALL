API CALL - POSTMAN 

SHORT DESCRIPTION
MY PROJECT CONTAINS AN API - POSTMAN THAT CAN DO REQUEST LIKE GET , PUT , POST , PATCH AND DELETE.

SET UP DESCRIPTION 
1. FIRST CREATE PROJECT USING LARAVEL NEW "NAME" IN CMD.
2. GO TO TERMINAL CREATE A PHP ARTISAN MAKE:MODEL STUDENT ,PHP ARTISAN MAKE:MIGRATION CREATE_PERSONAL_ACCESS_TOKENS_TABLE, CREATE_STUDENT_TABLE AND CREATE AND ADD IN ROUTES API.PHP AND HTTP/CONTROLLER CREATE A STUDENTS.CONTROLLER
3. CREATE A TABLE IN STUDENTS USING PHP ARTISAN TINKER
4. OPEN POSTMAN
5. OPEN WORKSPACE / CONNETED GMAIL OR GITHUB
6. CREATE A COLLECTION
7. IN THE COLLECTION CREATE A REQUEST
8. GET THIS IS USE TO GET THE JSON FORMAT OR THE COMPONENTS OF AN STUDENTS
9. POST THIS POST ADD THE NAME COURSE AND THE EMAIL OF AN STUDENTS
10. PUT SELECT A STUDENT 1 OR ID AND PUT AN EMAIL OR NAME OR COURSE
11. PATCH THIS IS A ONE SELECTION UPDATE
12. DELETE YOU CAN DELETE STUDENTS BY ID OR ALL
13. RUN THE URL HTTP http://127.0.0.1:8000/api/students/
